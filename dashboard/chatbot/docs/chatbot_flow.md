## Key Takeaways

1. **The AI doesn't directly access the database** - Context is pre-fetched and injected into the system prompt
2. **Chart generation uses a special tag format** - `[CHART]...[/CHART]` that gets parsed after the AI responds
3. **The database is queried twice for charts** - Once to convert namesâ†’IDs, once to fetch historical data
4. **Conversation history persists** - Enabling follow-up questions and contextual responses
5. **The AI is instructed on what it can/cannot do** - Preventing it from promising actions it can't perform
6. **Cost efficient** - Only latest prices sent to API, historical data only used locally for charts
